#Читач Документів для Android

Легкий та потужний читач документів для Android, який підтримує більше 50 форматів файлів.

## 📱 Можливості

### Підтримувані формати:

#### 📄 Текстові документи
- `.txt`, `.md`, `.tex`, `.bib`, `.ini`, `.ris`, `.enw`, `.rmd`

#### 📝 Microsoft Office
- Word: `.doc`, `.docx`, `.docm`
- Excel: `.xls`, `.xlsx`, `.xlsm`
- PowerPoint: `.ppt`, `.pptx`

#### 📊 OpenDocument формати
- `.odt` (Writer)
- `.ods` (Calc)
- `.odp` (Impress)
- `.odf`

#### 📕 PDF та електронні книги
- `.pdf` - PDF документи
- `.epub` - ePub книги
- `.mobi`, `.azw`, `.fb2` - інші формати електронних книг (базова підтримка)

#### 📐 Інші текстові формати
- `.rtf` - Rich Text Format
- `.csv`, `.tsv` - табличні дані
- `.dif`, `.slk` - Data Interchange Format

#### 🗄️ Архіви
- `.zip` - ZIP архіви
- `.tar` - TAR архіви
- `.gz` - GZIP стиснення
- `.xz` - XZ стиснення
- `.rar`, `.7z`, `.iso` - базова підтримка

#### 💾 Формати даних
- `.json` - JSON файли
- `.xml`, `.svg` - XML та SVG
- `.yaml`, `.yml` - YAML файли

#### 🎨 Спеціалізовані формати
- `.wps` - Microsoft Works
- `.pages` - Apple Pages
- `.key` - Apple Keynote
- `.ai`, `.eps`, `.ps` - Adobe формати
- `.xps` - XML Paper Specification
- `.dwg`, `.dxf` - CAD формати
- `.nb`, `.ipynb` - Notebook формати
- `.p7s`, `.p7m`, `.sig` - цифрові підписи

## 🛠️ Використані бібліотеки

- **Apache POI 5.2.3** - читання Microsoft Office документів
- **PDFBox Android 2.0.27.0** - обробка PDF файлів
- **Markwon 4.6.2** - рендеринг Markdown
- **Gson 2.10.1** - парсинг JSON
- **Simple XML 2.7.1** - парсинг XML
- **EPUBlib 3.1** - читання EPUB книг
- **Apache Commons Compress 1.24.0** - робота з архівами
- **RTFParserKit 1.16.0** - підтримка RTF
- **OpenCSV 5.8** - обробка CSV файлів
- **SnakeYAML 2.2** - парсинг YAML

## 📋 Вимоги

- **minSdk**: 26 (Android 8.0 Oreo)
- **targetSdk**: 34 (Android 14)
- **Java**: 11
- **Дозволи**:
  - `READ_EXTERNAL_STORAGE` (Android 6-12)
  - `READ_MEDIA_DOCUMENTS` (Android 13+)
  - `INTERNET` (для потенційного завантаження файлів)

## 🚀 Встановлення та використання

### 1. Клонування репозиторію
```bash
git clone [URL_РЕПОЗИТОРІЮ]
cd DocumentReader
```

### 2. Відкриття в Android Studio
1. Відкрийте Android Studio
2. File → Open → виберіть папку проекту
3. Дочекайтесь синхронізації Gradle

### 3. Збірка та запуск
```bash
./gradlew build
```

Або натисніть "Run" в Android Studio.

## 💡 Як користуватися

1. Запустіть додаток
2. Надайте дозвіл на читання файлів (якщо запитає)
3. Натисніть кнопку "Відкрити документ"
4. Виберіть файл з файлового менеджера
5. Документ буде відображено в текстовому форматі

## 🔧 Конфігурація

### Додавання підтримки нових форматів

Щоб додати підтримку нового формату:

1. Додайте відповідну бібліотеку в `app/build.gradle`
2. Додайте обробку в `DocumentReader.java` у методі `readDocument()`
3. Додайте MIME тип в `MainActivity.openFilePicker()`

### Приклад додавання формату

```java
// В DocumentReader.java
case "новий_формат":
    return readНовийФормат(uri);

private String readНовийФормат(Uri uri) throws Exception {
    // Ваша логіка читання
}
```

## ⚠️ Відомі обмеження

1. **RAR, 7Z, ISO** - потребують додаткових нативних бібліотек
2. **Pages, Key, Numbers** - формати Apple, складні для парсингу
3. **DWG, DXF** - CAD формати, потребують спеціалізованих бібліотек
4. **MOBI, AZW** - формати Amazon Kindle з DRM захистом
5. **Великі файли** - можуть викликати проблеми з пам'яттю

## 🐛 Вирішення проблем

### Помилка "Meta-inf" при збірці
Додано налаштування `packagingOptions` в `build.gradle` для виключення конфліктуючих файлів.

### OutOfMemoryError при великих файлах
Розгляньте можливість потокового читання або розбиття файлу на частини.

### Дозволи не працюють
Переконайтеся, що надали дозволи в налаштуваннях Android вручну.
