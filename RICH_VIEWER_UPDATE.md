## 🎨 Приклади форматування

### DOCX документ:
```html
<h1>Заголовок 1</h1>
<h2>Заголовок 2</h2>
<p style="text-align:center">
  <strong>Жирний текст</strong>
  <em>Курсив</em>
  <u>Підкреслений</u>
</p>
<img src="data:image/png;base64,..." class="doc-image"/>
<table class="doc-table">
  <tr><td>Комірка 1</td><td>Комірка 2</td></tr>
</table>
```

### Excel таблиця:
```html
<h3>📊 Аркуш 1</h3>
<table class="excel-table">
  <tr><td>Назва</td><td>Значення</td></tr>
  <tr><td>Дані 1</td><td>100</td></tr>
</table>
```

### PDF:
```html
<div class="pdf-info">
  <h3>📄 PDF документ</h3>
  <p>Сторінок: 15</p>
</div>
<div class="pdf-page">
  <p class="page-number">Сторінка 1</p>
  <img src="data:image/png;base64,..." class="pdf-page-image"/>
</div>
```

## 🚀 Як працює

### Потік даних:

```
1. MainActivity
   └─ Користувач обирає файл
   
2. DocumentViewerActivity
   ├─ Отримує URI + назву файлу
   ├─ Показує ProgressBar
   └─ Запускає фоновий потік
   
3. RichDocumentReader
   ├─ Визначає тип файлу
   ├─ Читає з форматуванням
   ├─ Генерує HTML + CSS
   └─ Повертає готовий HTML
   
4. WebView
   ├─ Завантажує HTML
   ├─ Рендерить з CSS
   └─ Показує користувачу
```

### Обробка зображень:

```java
// 1. Витягування з DOCX
List<XWPFPictureData> pictures = document.getAllPictures();

// 2. Конвертація в Base64
String base64 = Base64.encodeToString(pictureData, Base64.NO_WRAP);

// 3. Вбудовування в HTML
<img src="data:image/png;base64,iVBORw0KG..." />
```

### Рендеринг PDF:

```java
// 1. Відкрити документ
PDDocument document = PDDocument.load(inputStream);

// 2. Створити renderer
PDFRenderer renderer = new PDFRenderer(document);

// 3. Рендерити кожну сторінку
Bitmap bitmap = renderer.renderImage(pageNum, 2.0f); // 2x якість

// 4. Конвертувати в Base64
String base64 = bitmapToBase64(bitmap);

// 5. Вбудувати в HTML
<img src="data:image/png;base64,..." />
```

## 🔧 Технічні деталі

### Продуктивність:
- PDF обмежено 10 сторінками для швидкості
- Зображення стискаються (PNG 90% якість)

### Сумісність:
- Android 8.0+ (API 26)
- Всі розміри екранів
- Портрет і ландшафт
- Планшети підтримуються
